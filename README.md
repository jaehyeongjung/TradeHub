# TradeHub  

> **A 24/7 crypto traders’ hub with real-time chat, live prices, and a vibrant community.**  
> 실시간 **채팅**, 실시간 **코인 시세**, **Top Trader의 선물 포지션 비율**,  
> 그리고 **커뮤니티**가 함께하는 24시간 크립토 트레이더 허브.  
>  
> ⚠️ **PC 버전만 지원** — 대부분의 트레이더는 PC 환경에서 시세를 확인하고 분석하기 때문에,  
> 초기 MVP 단계에서는 **데스크톱 UX 최적화**에 집중했습니다.

## Tech Stack  

| Category | Stack |
|-----------|--------|
| **Frontend** | [Next.js (App Router)](https://nextjs.org/), TypeScript, TailwindCSS |
| **Backend** | [Supabase](https://supabase.com/) (Postgres · Realtime · Auth · Storage) |
| **Infra / Deploy** | [Vercel](https://vercel.com/) (CI/CD, Edge Ready) |

---

## Features  

- 실시간 **코인 시세** 및 **Top Trader 포지션 비율** 시각화  
- Supabase **Realtime** 기반의 **실시간 채팅**  
- 사용자 **게시글 작성 및 피드백 시스템**  
- **코인 뉴스 대시보드** — 하루 단위로 최신 뉴스 자동 수집
