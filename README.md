- **Frontend:** [Next.js](https://nextjs.org/) (App Router), TypeScript, TailwindCSS  
- **Backend:** [Supabase](https://supabase.com/) (Postgres, Realtime, Auth, Storage)  
- **Infra & Deploy:** [Vercel](https://vercel.com/) (CI/CD, Edge Ready)
